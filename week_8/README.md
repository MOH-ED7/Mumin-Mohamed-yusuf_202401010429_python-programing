# Week 8 - Food Delivery System

## 1. Purpose of the Application
This is a modular Python command-line application built as part of a Food Delivery System. It collects customer information, processes food orders (Cake or Muffin), calculates costs including delivery fees and a 5% service charge, and generates a formatted receipt.

## 2. Tech Stack
*   **Language:** Python 3
*   **Concepts:** Modular Programming (separating logic into `main.py`, `customer.py`, and `receipt.py`)
