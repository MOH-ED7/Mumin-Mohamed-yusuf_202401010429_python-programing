# IT Helpdesk Ticket Registration System

## 1. Purpose of the Application
The IT Helpdesk Ticket Registration System is a modular Python application designed for City University Malaysia's IT Support team. It allows students to report technical issues (e.g., login issues, Wi-Fi connectivity, hardware problems), automatically assigns an IT technician based on issue priority, and prints a formatted helpdesk ticket.

## 2. Tech Stack
* **Language:** Python 3
* **Tools:** Git, GitHub, VS Code / IDE
